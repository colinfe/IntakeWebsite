Programmeer ervaringen

Software

Voor het programmeren van HTML, CSS, Javascript en web3.js gebruik ik de editor Sublime Text Editor. Dit vind ik een erg fijne editor omdat het mooi overzichtelijk is met allemaal kleurtjes.

Voor het programmeren van Java gebruik ik Eclipse. Deze editor gebruik ik omdat het erg simpel te gebruiken is.

Voor het programmeren van Android apps gebruik ik Android Studio. Omdat je hier een simpele Drag & Drop functie hebt om snel buttons of text in te plaatsen.

Voor het programmeren van Solidity gebruik ik Remix. Omdat er toendertijds er nog geen goeie editor was om Solidity te prorgammeren, het bevalt me nu prima dus schakel ik niet over.
