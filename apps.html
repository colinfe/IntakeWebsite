Welke programmeer talen worden gebruikt voor het maken van apps?

Android

Voor het maken van Android apps kun je verschillende programmeertalen gebruiken als C++, Kotlin. Maar Java is toch wel de bekenste voor het maken van Android apps. Er zijn ook erg veel libraries te verkrijgen zijn voor Java zodat de programmeur het simpeler krijgt. Omdat Java zo bekent is valt er ook erg veel over te vinden op internet. Daarom zou ik zelf ook Java gebruiken voor het programmeren van Android Apps.

iOS

Voor het maken van IOS apps kun je verschillende programmeertalen gebruiken als C++, Objective-C, Swift. Ik zelf zou Swift gebruiken, dit komt omdat dit simpelere code is om te leren en om te lezen. Ook is het sneller.
